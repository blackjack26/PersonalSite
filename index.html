Cognitive Release

This project is a Javascript text-adventure game completed for my Intro to Programming class at Marist College. The player is able to enter commands into a textfield and move their character about a map displayed on their screen. As the user types, the available commands highlight themselves based on the entered text. Depending on how the game is played, the end-game is different.

Graphic Narrative Database (In Progress)

This project is a web application that allows students to write articles about different graphic narratives, post them to the site, and search other articles written by students. Before the articles are published, a faculty member will read over the article and eiter reject or approve it. If rejected, the student must fix the issues and resubmit. The goal of the site is to be used for educational purposes and not a fan site.

The project is currently in development by myself and a team of 3 other students. While I am the project lead, everyone is assigned tasks each week. I have touched almost every part of the site for styling and to add PHP functionality.

Feedback Manager

This PHP web application is used for managing customer feedback. I created this project with Gary Coltrane for the Marist College Computer Society hackathon during the spring of 2016. The prompt was given by The Hartford to create an application where customers could give suggestions to the company and the administrators could review and implement (or not) the suggestions.

In our application, customers can log in and vote on existing suggestions or create their own. If a duplicate suggestion is created, it can be marked by the admin. Administrators can have conversations through the app within each suggestion (chat room functionality). In the end, they can choose to either implement or not implement the suggestion. There is also statistics about the data and other management options available.
